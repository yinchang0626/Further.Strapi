# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Further.Strapi integration library
- Complete Strapi v5 API support
- Collection Type Provider with advanced querying
- Single Type Provider for global content
- Media Library Provider with upload and management capabilities
- Polymorphic component handling for dynamic zones
- Smart serialization with system field filtering
- Comprehensive test suite with integration and unit tests
- Fluent query builders for filtering, sorting, and population
- Type-safe C# interfaces for all operations

### Features
- **Collection Types**: Full CRUD operations with advanced querying
- **Single Types**: Get and update operations for global content
- **Media Library**: File upload, metadata management, and organization
- **Dynamic Components**: Automatic serialization/deserialization of polymorphic components
- **Query Building**: Fluent API for complex filtering, sorting, and population
- **Performance**: Optimized HTTP client management and efficient data handling

### Technical Details
- Built on .NET 9.0 and ABP Framework 9.3.5
- Comprehensive error handling and logging
- Extensive test coverage with real Strapi integration tests
- Clean architecture with separated concerns
- Full async/await support throughout

## [1.0.0] - 2025-01-XX

- Initial stable release

- Complete documentation and examples
